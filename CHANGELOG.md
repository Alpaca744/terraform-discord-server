# Changelog

All notable changes to this module are documented here. This module follows
[Semantic Versioning](https://semver.org/).

## [0.1.1] - 2026-06-21

### Changed
- Constrain the `alpaca744/discord` provider to `>= 0.1.0, < 1.0.0` (root,
  submodules, and examples) so the module composes cleanly while guarding
  against a future breaking `1.0.0`.

## [0.1.0] - 2026-06-21

### Added
- Initial release: opinionated Discord server baseline composing the `roles`,
  `channels`, `moderation`, and `onboarding` submodules.
